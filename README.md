# Nokta Notes

Nokta is a local-first note editor with a built-in MCP server for AI-assisted writing and research. Everything is stored in a single `.nokta` file.

##### ℹ️ [Latest release & download](https://github.com/BadgerSoft-NL/NoktaNotes/releases)
##### 🌐 [Website & MCP setup guide](https://noktanotes.badgersoft.nl/)
##### 🔎 [Download the manual (.nokta file)](https://github.com/BadgerSoft-NL/NoktaNotes/raw/main/Manual.nokta)

## Key Features

- Unlimited nested notes with drag-and-drop organization
- Rich text, Markdown, and plain text / code editor
- Full-text search across your entire notebook
- Synopsis per note for writers and worldbuilders
- 10 built-in themes, customizable font and size
- EPUB export for notes and folders
- Workflow prompt templates in context menus
- Built-in MCP server: let AI assistants read and write your notes directly
- No account, no cloud, no subscription

## MCP Server

Nokta has a built-in MCP server. AI assistants like Claude can connect to your notebooks and read, search, create, and edit notes on your behalf.

See the [MCP setup guide](https://noktanotes.badgersoft.nl/mcp) on the website to get started.
